404 - Page Not Found

The page you are looking for is missing.

How to Fix It Manually

Check the URL above and make sure it ends with:

/home.html

Example fix:

If your URL is:

https://yourusername.github.io/e-shoe/abc

Change it manually to:

https://yourusername.github.io/e-shoe/home.html

This will fix the error and open the homepage.
